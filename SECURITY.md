# Security Policy

## Reporting a Vulnerability

**Do not report security vulnerabilities through public GitHub issues.**

Email us at **security@quvi.ai** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact assessment

We will acknowledge receipt within 48 hours and aim to resolve confirmed
vulnerabilities within 14 days.

## Scope

- API key storage and handling within the add-on
- Network communication with the QUVIAI API
- Blender add-on sandboxing

## Out of Scope

- The QUVIAI backend API itself — contact us directly for backend issues
- Vulnerabilities in Blender itself
